# 产品需求概览

## 产品定位

Loeyae Boot Framework 是一套企业级 Java 后端开发框架，基于 Spring Boot 3.5.7 + Java 21，为开发团队提供开箱即用的基础设施能力，帮助快速构建安全、稳定、可扩展的业务系统。

## 核心业务域

| 业务域 | 说明 | 优先级 |
|--------|------|--------|
| 基础设施 | 统一响应、分页、异常体系、工具类 | P0 |
| Web 服务 | API 规范、全局异常处理、安全防护（XSS/CORS）、请求加密 | P0 |
| 数据访问 | CRUD 封装、多租户、数据权限、字段加密、数据审计 | P0 |
| 安全认证 | JWT 认证、RBAC 权限、会话管理 | P0 |
| 缓存服务 | 本地缓存、分布式缓存、二级缓存 | P0 |
| 消息队列 | 统一消息抽象、Kafka/RabbitMQ/Redis Stream 实现、消息审计 | P1 |
| 定时任务 | Quartz 动态任务管理 | P1 |
| 扩展能力 | 邮件、CMS、Feign 调用、许可证、接口签名、数据脱敏 | P2 |

## 全局约束

- Java 21 为最低版本，不支持向下兼容
- 依赖 Spring Boot 3.5.7 生态
- 所有模块通过 BOM 统一版本管理
- 模块化设计，业务应用按需引入 starter

## 非功能需求概要

- 性能：框架层面零额外开销，不引入不必要的拦截链
- 安全：默认安全（XSS 过滤、CSRF 防护、参数校验）
- 可用性：开箱即用，最小配置即可启动
