局部改动 #3 规则详情页（多状态） 关联: US-03, US-04 新窗口打开 ↗

推导说明

  • 基准状态选择：选择「已上线」状态作为基准，因为信息量最丰富
  • 操作按钮差异：5种状态各自有不同的操作组合，通过条件表列出

业务规则

  • 打回规则：已通过可打回到草稿，需填写打回原因
  • 上线校验：上线前校验关联商品是否就绪
