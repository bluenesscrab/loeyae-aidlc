## 平台边界

Kiro、Claude Code、OpenCode 是产品入口，不是业务项目属性，也不是 `ssot-api` 的直接 Consumer。平台凭据或 Hook 差异只能影响调用方式和展示；若改变 RoleIntent、资料选择、引用或 state v2 语义，则视为一致性失败。
