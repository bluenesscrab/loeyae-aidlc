# 产品级契约索引

**职责**：定义产品级 Inception 如何记录跨模块、跨服务和外部系统契约。兼容性、消费者状态和验证规则统一加载 `common-contract-governance.md`，本文件不复制其细节。

## 执行条件

存在多模块、跨进程调用、异步事件、前后端接口或外部系统交换时执行。纯单模块进程内调用可在应用设计中定义，不创建产品级契约索引。

## 边界区分

- **进程内模块契约**：可在应用设计中记录完整方法签名，权威事实通常是代码。
- **跨进程服务契约**：以项目采用的机器可读接口、事件 Schema 或生成源为权威事实。
- **外部系统契约**：以对方规范、适配层契约或已确认交换定义为权威事实。
- **共享技术能力**：可由公共包提供；业务 Entity、数据库模型和领域对象不默认跨服务共享。

服务是独立部署和数据所有权边界；模块是服务或单体内部的逻辑边界。两者不得混用。

## `contracts.md` 产物

创建或更新 `docs/aidlc/product/contracts.md`：

```markdown
# 产品级契约索引

## 契约总览
[只展示跨边界方向的 ASCII 图]

## 契约清单
| 契约 ID | 类型 | 提供方 | 权威来源 | 版本/策略 | 兼容状态 | Owner |
|---------|------|--------|----------|-----------|----------|-------|

## 消费者状态
| 契约 ID | 消费者 | 影响 | 状态 | 验证证据 |
|---------|--------|------|------|----------|

## 决策与约束
- 数据所有权、鉴权、错误语义、幂等和版本并存决策的引用

## 变更日志
| 时间 | 契约 ID | 变更摘要 | 兼容性 | 影响消费者 | 状态 | CR/证据 |
|------|---------|----------|--------|------------|------|---------|
```

存在机器可读权威来源时，`contracts.md` 只保存索引、决策和状态，不复制完整字段。不存在机器来源时标记“文档契约”，不得声称已自动兼容校验。

## 在线资料契约

当产品通过 Provider 向 AI-DLC 或其他 Consumer 提供项目资料能力时，`contracts.md` 还必须覆盖以下传输无关语义：项目解析；资料、修订与片段读取；项目内检索及显式降级；上下文包；逆向说明上传；资料版本/片段到正式文档章节的引用与血缘写回；解析/索引状态；统一失败语义。

- 文档契约采用 Provider-first：先确定唯一 Provider 权威、Owner 和候选版本，再逐 Consumer 登记 `待评估/待适配/已验证` 状态。
- Consumer 只引用契约 ID 和语义，不复制 Provider 业务规则；平台适配层不得直接形成第二套契约。
- 字段级机器 Schema、鉴权、项目隔离、幂等、版本并存、错误码和兼容窗口必须在 I12 冻结；此前一律不得声称运行兼容已验证。
- Provider 失败或索引不可用时必须显式报告状态和降级来源，不得把降级结果表述为完整结果或事实不存在。
- 未配置 Provider 的 Legacy 流程不得产生额外远程调用或阻断。

## 创建流程

1. 从服务目录和运行时依赖基线识别提供方与消费者。
2. 定位每项契约的机器事实来源；冲突或缺失列入待确认项。
3. 按 `common-contract-governance.md` 记录兼容策略和逐消费者状态。
4. 将数据所有权、跨边界禁止直接访问内部实现等约束写入决策引用。
5. 经用户确认后作为模块级 Inception 和并行开发的共享索引。

## 变更与恢复

契约变化必须进入 CR：
1. 更新权威契约来源；
2. 基于消费者反向索引评估影响；
3. 在变更日志逐项记录消费者状态；
4. 只有消费者完成适配并有执行证据后才标记“已验证”。

恢复模块或服务工作时，只加载契约索引和当前消费者相关的机器契约切片。发现 `待评估` 或 `待适配` 项时，先提示并按 CR 计划处理，不得直接把全局记录改为已同步。

## 循环依赖处理

发现潜在循环时，依据业务边界选择接口反转、异步解耦或重新划分职责。不得默认把双方业务对象提取到“基础模块”，也不得通过共享数据库绕过契约。

## 完成标准

- [ ] 跨边界契约与进程内方法已区分
- [ ] 每项契约有权威来源或“文档契约”标记
- [ ] 提供方、消费者、Owner 和数据边界明确
- [ ] 每个消费者有独立适配与验证状态
- [ ] 机器契约与索引无冲突，或冲突已阻断并待确认
- [ ] state.md 记录契约索引路径、新鲜度和未完成消费者
